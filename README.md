# рЃЎрЃЋрЃўрЃарЃўрЃА рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃћрЃЉрЃўрЃА рЃфрЃ«рЃарЃўрЃџрЃў - Schedule App

рЃЎрЃЋрЃўрЃарЃўрЃА рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃћрЃЉрЃўрЃА рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃў рЃЋрЃћрЃЉ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ Google Drive рЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃўрЃќрЃљрЃфрЃўрЃўрЃЌ.

## рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў

РюЁ рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ, рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ рЃЊрЃљ рЃгрЃљрЃерЃџрЃљ  
РюЁ рЃАрЃљрЃљрЃЌрЃћрЃЉрЃўрЃА рЃЊрЃљ рЃ«рЃљрЃюрЃњрЃарЃФрЃџрЃўрЃЋрЃЮрЃЉрЃўрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљ  
РюЁ рЃЮрЃЌрЃљрЃ«рЃўрЃА рЃюрЃЮрЃЏрЃарЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ  
РюЁ рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃљ рЃћрЃарЃЌрЃў рЃЊрЃдрЃўрЃЊрЃљрЃю рЃЏрЃћрЃЮрЃарЃћрЃќрЃћ  
РюЁ рЃцрЃўрЃџрЃбрЃарЃљрЃфрЃўрЃљ рЃЊрЃдрЃћрЃћрЃЉрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ  
РюЁ рЃАрЃбрЃљрЃбрЃўрЃАрЃбрЃўрЃЎрЃљ  
РюЁ Google Drive рЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃўрЃќрЃљрЃфрЃўрЃљ  
РюЁ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ localStorage-рЃерЃў  

## Google Drive рЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ

рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА Google Drive API-рЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃдрЃарЃБрЃЉрЃћрЃџрЃерЃў рЃерЃћрЃАрЃљрЃюрЃљрЃ«рЃљрЃЊ. рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃюрЃљрЃЉрЃўрЃ»рЃћрЃЉрЃў:

### 1. Google Cloud Console-рЃерЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ

1. рЃерЃћрЃЊрЃў [Google Cloud Console](https://console.cloud.google.com/)
2. рЃерЃћрЃЦрЃЏрЃћрЃюрЃў рЃљрЃ«рЃљрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃљрЃю рЃљрЃўрЃарЃЕрЃўрЃћ рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў
3. рЃЊрЃљрЃљрЃГрЃўрЃарЃћ "Enable APIs and Services"
4. рЃЏрЃЮрЃФрЃћрЃЉрЃюрЃћ рЃЊрЃљ рЃЕрЃљрЃарЃЌрЃћ **Google Drive API**

### 2. OAuth 2.0 Credentials-рЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ

1. рЃгрЃљрЃЊрЃў **APIs & Services** Рєњ **Credentials**
2. рЃЊрЃљрЃљрЃГрЃўрЃарЃћ **"Create Credentials"** Рєњ **"OAuth client ID"**
3. рЃЌрЃБ рЃърЃўрЃарЃЋрЃћрЃџрЃљрЃЊ рЃЦрЃЏрЃюрЃў, рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃљрЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃЮ OAuth consent screen:
   - User Type: **External**
   - App name: `Schedule App`
   - User support email: рЃерЃћрЃюрЃў email
   - Developer contact: рЃерЃћрЃюрЃў email
   - рЃЊрЃљрЃљрЃГрЃўрЃарЃћ **Save and Continue**
   - Scopes: рЃњрЃљрЃЏрЃЮрЃбрЃЮрЃЋрЃћ
   - Test users: рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћ рЃерЃћрЃюрЃў Google email
   - рЃЊрЃљрЃљрЃГрЃўрЃарЃћ **Save and Continue**

4. OAuth client ID-рЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ:
   - Application type: **Web application**
   - Name: `Schedule App Web Client`
   - Authorized JavaScript origins:
     - `http://localhost`
     - `http://localhost:8080`
     - `file://` (рЃЌрЃБ рЃцрЃљрЃўрЃџрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃ«рЃАрЃюрЃў)
   - рЃЊрЃљрЃљрЃГрЃўрЃарЃћ **Create**

5. **рЃЊрЃљрЃљрЃЎрЃЮрЃърЃўрЃарЃћ Client ID** (рЃЏрЃљрЃњ: `123456789-abc.apps.googleusercontent.com`)

### 3. API Key-рЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ

1. **Credentials** рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ рЃЊрЃљрЃљрЃГрЃўрЃарЃћ **"Create Credentials"** Рєњ **"API key"**
2. **рЃЊрЃљрЃљрЃЎрЃЮрЃърЃўрЃарЃћ API Key**
3. (рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃў) рЃЊрЃљрЃљрЃГрЃўрЃарЃћ **Restrict Key**:
   - API restrictions Рєњ Restrict key
   - рЃЏрЃЮрЃюрЃўрЃерЃюрЃћ **Google Drive API**
   - рЃЊрЃљрЃљрЃГрЃўрЃарЃћ **Save**

### 4. рЃЎрЃЮрЃЊрЃерЃў Credentials-рЃўрЃА рЃЕрЃљрЃАрЃЏрЃљ

рЃњрЃљрЃ«рЃАрЃћрЃюрЃў `schedule-app.html` рЃцрЃљрЃўрЃџрЃў рЃЊрЃљ рЃўрЃърЃЮрЃЋрЃћ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃ«рЃљрЃќрЃћрЃЉрЃў:

```javascript
const CLIENT_ID = 'YOUR_CLIENT_ID_HERE.apps.googleusercontent.com';
const API_KEY = 'YOUR_API_KEY_HERE';
```

рЃЕрЃљрЃљрЃюрЃљрЃфрЃЋрЃџрЃћ:
- `YOUR_CLIENT_ID_HERE.apps.googleusercontent.com` Рєњ рЃерЃћрЃюрЃў Client ID
- `YOUR_API_KEY_HERE` Рєњ рЃерЃћрЃюрЃў API Key

### 5. рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ

**рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў 1 - рЃЏрЃљрЃарЃбрЃўрЃЋрЃў HTTP рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃЌ (рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃБрЃџрЃў):**

рЃЌрЃБ рЃњрЃљрЃЦрЃЋрЃА Python:
```bash
# Python 3
python -m http.server 8080

# Python 2
python -m SimpleHTTPServer 8080
```

рЃерЃћрЃЏрЃЊрЃћрЃњ рЃњрЃљрЃ«рЃАрЃћрЃюрЃў рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃерЃў: `http://localhost:8080/schedule-app.html`

**рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў 2 - VS Code Live Server-рЃўрЃЌ:**
1. рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћ Live Server extension
2. Right-click Рєњ Open with Live Server

**рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў 3 - рЃцрЃљрЃўрЃџрЃўрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃњрЃљрЃ«рЃАрЃюрЃљ:**
рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃЮрЃарЃ»рЃћрЃа рЃЊрЃљрЃљрЃгрЃЎрЃљрЃърЃБрЃюрЃћ `schedule-app.html` рЃцрЃљрЃўрЃџрЃќрЃћ (рЃћрЃА рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃљрЃа рЃўрЃЏрЃБрЃерЃљрЃЮрЃА рЃДрЃЋрЃћрЃџрЃљ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃерЃў рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃўрЃА рЃерЃћрЃќрЃдрЃБрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮ)

## рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ

### рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ
1. рЃљрЃўрЃарЃЕрЃўрЃћ рЃЊрЃдрЃћ
2. рЃЊрЃљрЃљрЃГрЃўрЃарЃћ "+ рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ"
3. рЃерЃћрЃљрЃЋрЃАрЃћ рЃАрЃљрЃљрЃЌрЃў, рЃ«рЃљрЃюрЃњрЃарЃФрЃџрЃўрЃЋрЃЮрЃЉрЃљ, рЃАрЃљрЃњрЃљрЃюрЃў, рЃЮрЃЌрЃљрЃ«рЃў
4. рЃЊрЃљрЃљрЃГрЃўрЃарЃћ "рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ"

### рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃўрЃА рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ
1. рЃЊрЃљрЃљрЃГрЃўрЃарЃћ "рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ" рЃдрЃўрЃџрЃљрЃЎрЃА рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃќрЃћ
2. рЃерЃћрЃфрЃЋрЃљрЃџрЃћ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ
3. рЃЊрЃљрЃљрЃГрЃўрЃарЃћ "рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ"

### рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃљ
1. рЃЊрЃљрЃљрЃГрЃўрЃарЃћ ­ЪЊІ рЃдрЃўрЃџрЃљрЃЎрЃА рЃЊрЃдрЃўрЃА рЃЌрЃљрЃЋрЃАрЃљрЃарЃЌрЃерЃў
2. рЃљрЃўрЃарЃЕрЃўрЃћ рЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃў рЃЊрЃдрЃћрЃћрЃЉрЃў
3. рЃЊрЃљрЃљрЃГрЃўрЃарЃћ "рЃЎрЃЮрЃърЃўрЃарЃћрЃЉрЃљ"

### Google Drive рЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃўрЃќрЃљрЃфрЃўрЃљ
1. рЃЊрЃљрЃљрЃГрЃўрЃарЃћ "­Ъћљ Google-рЃўрЃЌ рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ"
2. рЃЊрЃљрЃљрЃЊрЃљрЃАрЃбрЃБрЃарЃћ рЃгрЃЋрЃЊрЃЮрЃЏрЃљ
3. "Ргє№ИЈ рЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ Drive-рЃќрЃћ" - рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃюрЃљрЃ«рЃљрЃЊ
4. "РгЄ№ИЈ рЃЕрЃљрЃЏрЃЮрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ Drive-рЃЊрЃљрЃю" - рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃљрЃдрЃАрЃљрЃЊрЃњрЃћрЃюрЃљрЃЊ
